Handoff — back to Figma & to dev

To Figma (ready-made): ⬇ litro-desktop-redesign.fig — a native Figma file with all three screens as editable frames (layers, texts, photos, exact sizes; single-line texts are auto-width). Drag it onto the Figma home screen / “Import” — done. Fonts used: Outfit + DM Sans (both in Figma’s Google Fonts catalog).

Alternative route: the html.to.design plugin (already used by the team for “LITRO Master Pages”) — capture these live pages: home.html, listing.html, hotel.html from this site’s URL, or via its Chrome extension.

To dev: litro.css is the token sheet (colors, radii, shadows, type as CSS variables — all values in px) and specs.html is the measurement spec (component-by-component). The HTML files themselves are reference implementations — paddings, gaps and font sizes can be lifted 1:1.
Scope note. These are the 3 priority templates (per the priorities noted in the LITRO Figma: booking flow → hotel details → listing). Checkout & thank-you desktop screens follow the same Szallas sandbox layouts with this skin — to be produced next once this direction is confirmed. Photos are placeholders taken from the sandbox assets; swap for real LITRO property photos in production.
